# iOS app

SwiftUI app, iOS 17 minimum, generated by [xcodegen](https://github.com/yonaskolb/XcodeGen)
from `ios/project.yml`. **The `.xcodeproj` is generated** — don't check it
into git, regenerate with `xcodegen` after editing `project.yml`.

## Layout

```
ios/
├── project.yml
└── Sources/Lastscrew/
    ├── App.swift                ← @main + NavigationStack wiring
    ├── AppRouter.swift          ← AppRoute enum + ObservableObject path
    ├── Theme.swift              ← colors, fonts, CardStyle modifier
    ├── Models/APIModels.swift   ← Codable mirror of the Worker types
    ├── Networking/APIClient.swift
    ├── Camera/
    │   ├── CameraController.swift   ← AVFoundation session + capture
    │   └── CameraPreviewView.swift  ← UIViewRepresentable preview
    ├── ViewModels/
    │   ├── OfferViewModel.swift
    │   └── PackagingViewModel.swift
    ├── Screens/
    │   ├── ManageItemView.swift
    │   ├── ReturnChooserView.swift
    │   ├── OfferRevealView.swift
    │   ├── EarningsBreakdownView.swift
    │   ├── AcceptHostView.swift
    │   ├── PackagingCameraView.swift

        ├── WayfairHome.imageset
        ├── WayfairManageItem.imageset

```

## Navigation graph

```
ManageItemView (root)
  └─ ReturnChooserView
       └─ OfferRevealView ── EarningsBreakdownView (subview)
            └─ AcceptHostView
                 └─ PackagingCameraView
                      └─ PackagingResultView
                           └─ HostDashboardView
```

Driven by `AppRouter.path: [AppRoute]`. Each push appends a route; the root
view's `.navigationDestination(for: AppRoute.self)` dispatches.

## Configuration

The Worker base URL is read from the `LASTSCREW_API_BASE` Info.plist key
(default `http://127.0.0.1:8787`), with an override via the same-named scheme
environment variable. For a physical device test, set the env var to your
Mac's LAN IP.

## Camera permissions

`NSCameraUsageDescription` is set via `project.yml` and ends up in the
generated Info.plist. The Camera flow degrades to `PhotosPicker` on the
Simulator since it has no camera.

## Build steps

```bash
sudo xcode-select -s /Applications/Xcode.app/Contents/Developer   # one-time
cd ios
xcodegen generate
open Lastscrew.xcodeproj
# Cmd+R into the iPhone 15 Pro simulator.
```

## Testing on a real device

1. In Xcode, select your device.
2. Set a development team in `Signing & Capabilities`.
3. Set the `LASTSCREW_API_BASE` env var in the Scheme to
   `http://<your-mac-ip>:8787` and make sure the Mac and phone are on the
   same network.
4. The first Worker call may fail — accept the local network prompt and retry.
